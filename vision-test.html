PROJECT 03 / 实用工具

标准对数视力表
E视力测试

基于 GB 11533-2011《标准对数视力表》国家标准开发的在线视力自测工具。纯前端实现，浏览器直接打开即可使用，无需安装任何依赖。将标准 E 字视力表数字化，用户只需一台电脑显示器，选择合适的测试距离，即可在家中完成一次符合国家标准的视力自测。

BACKGROUND

为什么做这个项目

将国家标准视力表搬到浏览器里，让每个人都能随时随地进行一次有据可依的视力自测。

01
纸质视力表不便携带，数字化后随时可用，手机电脑即可测试。
02
多数在线视力测试缺乏标准依据，本项目严格遵循 GB 11533-2011 国标。
03
支持多种测试距离，适配从屏幕前近距离到远距离的不同使用场景。
04
纯前端实现，数据不离开用户设备，保护隐私，无需注册无需联网。

FEATURES

核心功能

完整标准视力表
14 个标准级别（4.0~5.3，对应小数视力 0.1~2.0），每级按屏幕空间动态排列 E 字视标，笔划宽度与缺口均为边长的 1/5，符合 1 分视角标准。
精准测试体系
红色箭头实时指向当前待测 E 字，高亮框标记测试视标位置。每级答对 80% 以上晋级，否则结束测试，结果自动判定视力等级并给出建议。
灵活距离调节
支持 40cm~90cm（每 10cm 一档）近距离与 1m~6m（每米一档）远距离，切换距离后视力表按国标比例自动缩放，适配屏幕前近距离使用场景。
多种操作方式
鼠标点击方向按钮、键盘方向键（↑↓←→）、键盘 WASD 快捷键，三种操作方式自由选择，适配不同使用习惯。
CALCULATION

视标尺寸计算

依据 GB 11533-2011 标准，5.0 级（小数视力 1.0）在 5 米标准距离下的 E 字边长为 7.272mm。各级别视标尺寸严格按以下公式计算。

SIZE FORMULA
边长(mm) = ( 7.272 / 小数视力 ) × ( 测试距离 / 5 )

PROCESS

使用流程

STEP 01

用浏览器打开 vision_test.html，无需安装任何依赖。
STEP 02

CODE

核心代码

E 字视标的 Canvas 矢量绘制 — 3 条横杠 + 1 条竖杠，笔划宽度为边长的 1/5，通过旋转实现四个朝向。这是整个项目最核心的绘制函数。

// vision_test.html — drawE() 函数
function drawE(ctx, cx, cy, size, direction) {
  const bar = size * 0.2;  // 笔划宽度 = 边长 × 1/5
  const gap = size * 0.2;  // 缺口宽度 = 边长 × 1/5

  ctx.save();
  ctx.translate(cx, cy);

  let rotation = 0;
  switch (direction) {
    case 'up':    rotation = -Math.PI / 2; break;
    case 'down':  rotation =  Math.PI / 2; break;
    case 'left':  rotation =  Math.PI;       break;
    case 'right': rotation =  0;             break;
  }
  ctx.rotate(rotation);
  ctx.fillStyle = '#1e293b';

  // 竖杠（左侧）
  ctx.fillRect(-size / 2, -size / 2, bar, size);
  // 上横杠
  ctx.fillRect(-size / 2 + bar, -size / 2, size - bar, bar);
  // 中横杠（右侧有缺口）
  ctx.fillRect(-size / 2 + bar, -bar / 2, size - bar - gap, bar);
  // 下横杠
  ctx.fillRect(-size / 2 + bar, size / 2 - bar, size - bar, bar);

  ctx.restore();
}
